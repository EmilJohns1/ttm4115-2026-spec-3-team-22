From drone:
1. status (GPS, Battery, Speed)  // what channel

From dock:
1. task_assignment (coordinates?) // to drone, on drone's channel
2. arrival_confirmation () // sth from client
3. drone_assignment (drone_ID) // to backend
4. assignment_failed (code)  // to backend

From backend:
1. assignment_request(coordinates) // to dock

.\protoc-25.9-win64\bin\protoc.exe --python_out=. messages.proto


Channel tree

delivery-system/
    drone/
        <droneID>/
            status
                # status (drone to backend)
            assignment
                # task assignment (dock to drone)
            confirmation
                # arrival confirmation (backend to drone, confirm to fly back)
            readiness
                # battery (drone to dock)
    drones/
        readiness
            # empty (dock to all drones)
    management/
        request
            # assignment request (backend to dock)
        assignment
            # drone assignment (dock to backend)
        failure
            # assignment failed (dock to backend)
